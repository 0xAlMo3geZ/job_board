# Job Board (Django/Nuxt.js)

a job board Web application using Django and Nuxt 3 (Vue 3). and Other technologies such as Tailwind, Pinia, Django Rest Framework.

# Project Structure

```
.
├── backend
├── frontend
│   ├── app.vue
│   ├── assets
│   ├── node_modules
│   ├── nuxt.config.ts
│   ├── package.json
│   ├── package-lock.json
│   ├── public
│   ├── server
│   ├── tailwind.config.js
│   └── tsconfig.json
├── README.md
└── requirements.txt
```
